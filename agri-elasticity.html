01 七维评分定义 SCORE 1–5 · 数值越高 = 弹性贡献越大
D1
供给集中度
前两大产国出口份额。集中度越高,单点冲击放大越剧烈。
代理:HHI / 出口前2占比
D2
天气敏感度
单产对温度/降雨/极端事件的弹性。霜冻、干旱、ENSO 周期。
代理:历史单产标准差/事件频率
D3
库销比紧度
全球期末库存/年消费。库存厚 = 缓冲器,库存薄 = 价格弹性放大。
代理:WASDE Stocks-to-Use
D4
替代刚性
下游难以切换替代品的程度。咖啡/可可几乎无替代,植物油彼此可替。
代理:消费习惯+技术工艺锁定
D5
政策干预风险
出口禁令/配额/最低收购价/储备投放/关税。方向和阈值比分数更重要——托价≠压价。
方向见 §04 图谱
D6
金融化深度
总持仓与基金净持仓占比。深 = 容纳大资金,薄 = 单边波动放大。
注:薄合约≠低弹性,反而事件放大
D7
地缘物流暴露
关键运输节点依赖度。黑海/马六甲/巴拿马/红海/密西西比。
代理:出口走航必经节点数

    05 情景叠加:Hormuz 海峡封闭 D7 只反映平时状态;极端封锁须作为独立叠加层
    方法论诚实 · Method Note

    七维框架中 D7 地缘物流 得分基于平时状态——常规航运路径、政治敏感度、主产国分散度。但 Hormuz 这类"极端封锁"不是渐变,是跳变,框架对此结构性盲视。解法不是修改 D7 权重——而是把封锁作为独立情景叠加层,给每品种一个 Δ 增量(0 ~ +3),在框架本体之外单独看。

    核心传导链有三:(1) 原油飙升 → 生柴掺混经济性抬升 → 油脂类直接受益;(2) 氮肥/尿素/磷肥成本暴涨 → 种植成本重估 → 谷物跟涨;(3) 亚洲能源补库 → LNG 紧张+红海/印度洋保险费溢价 → 远距离航运品种到岸溢价。弱相关品种靠"替代被压缩/非路径/国内闭环定价"三种机制被隔绝在外。

    Note · 不要把 Δ 当价格目标

    Δ 增量是框架下的结构性估计,非逐笔价格测算。实际落地需叠加四个变量:(a) 封锁持续时长——72 小时短闭锁与 30 天+ 封锁的传导量级差 10 倍;(b) 美方 SPR 释放节奏与 OPEC+ 补位意愿;(c) 中东其他产油国(沙特/阿联酋)物理补位能力;(d) 亚洲 LNG 买家(日/韩/中)合约覆盖比例与斋月/冬季取暖季节错位。另:此 Δ 与 D5 政策开关存在交互——棕榈/原糖等 direct 品种若触发 CPI 恐慌会被产国反手禁运,Δ 上限可达 +4,但此时已进入框架失效区,不在本表覆盖。

    06 情景叠加:ENSO · 厄尔尼诺 / 拉尼娜 D2 天气维度的周期性重构 · 全球传导
    当前状态 · Current Status

    赤道太平洋 2025–26 冬季弱拉尼娜,正过渡到 ENSO 中性。NOAA CPC 最新判定:4–6 月中性概率 80%;5–7 月厄尔尼诺形成概率 61%,大概率持续至 2026 年底,1/3 概率为"强"等级(Niño 3.4 ≥ +1.5°C)。

    这意味着 D2 天气维度在 2026 H2 进入厄尔尼诺叠加状态——印尼/马来干旱风险抬头(棕榈利多)、南美洲南部多雨(大豆供给双刃)、印度季风减弱(糖利多)、澳洲东部小麦减产、美中西玉米授粉期反而温和(玉米利空)。

    08 框架失效场景 · 何时不要用此分数排序 事件型、政策反转型、流动性悬崖型
    CASE A · 事件单点放大

    苹果 AP / 鸡蛋 JD

    总分中等,但天气与产能周期是单点高弹性变量。清明霜冻一个事件可使期价 30%+ 跳变,框架的均匀加权会显著低估。建议叠加事件日历单独看 D2 / D5。

    CASE B · 持仓薄反向放大

    橙汁 OJ / 红枣 CJ

    D6 金融化得分低本应是减分项,但极薄持仓在事件来临时反而无对手盘 → 涨停连板。OJ 在佛州黄龙病+飓风背景下 2023–24 创历史新高即此机制。低分品种需做反向解读。

    CASE C · 政策反转

    中国玉米 / 棉花 / 白糖

    D5 高分品种存在国储这把双向开关。收储推涨、抛储压价,方向反转时框架打分稳定但价格剧变。需配 收储节奏 / 拍卖底价 / 进口配额释放 三路独立信号。

    CASE D · 替代矩阵共振

    豆油 / 棕榈油 / 菜籽油

    三油 D4 替代刚性都低,但弹性需作为板块共振分析,不是单品种。当生柴政策/出口禁令切换主导逻辑,板块价差(P-Y / OI-Y)比单边更具结构性机会。

    CASE E · 国内孤品流动性

    苹果 / 红枣 / 生猪 / 鸡蛋

    无外盘对照,但流动性受国内政策+主力席位影响显著。框架适合发现弹性方向,但持仓集中度/单一席位风险需独立监控。

    CASE F · 地缘加速

    小麦 / 葵花籽油 / 玉米

    D7 地缘暴露评分基于平时状态。一旦黑海/红海发生主动封锁,这些品种弹性会从中段跳变到顶段,框架是滞后指示器,需配Hormuz式情景叠加。
