合约基于什么交易？ 合成定价 vs 实物代币 · 必读
⚙️ 一句话说清楚
币安 / 欧意 / Bitget 上的代币股永续合约，绝大多数 不是基于链上代币， 而是基于交易所自建的 "合成指数"（Synthetic Index）—— 由若干个传统券商 / 做市商 报价聚合而成的美股价格快照，乘以合约面值，按 USDT 现金交割。 只有 Bitget 与少数交易所的 xStocks 现货 才是 Backed Finance 在 Solana 链上发行的 1:1 实物背书代币（持牌托管商持有真股票）。这两条路径的风险与价格行为完全不同。
PATH A · 永续合约
合成指数 + USDT 现金交割
交易所自建价格指数（盘中：聚合多家券商；盘后：上一交易日收盘 + 隐含价/做市商报价）
PATH B · xStocks 现货
Backed Finance 链上代币
Solana 上 1:1 背书代币，瑞士持牌托管商持有真股票，可链上转账，可申赎（机构）
PATH C · 杠杆现货（保证金）
借币做空 + 现货持仓
基于 Path B 现货，加上交易所的借贷池。借币利率 ≠ 资金费率，需单独看
📊
指数计算（Path A）
美股交易时段（NYSE 21:30-04:00 UTC+8）：聚合多家做市商 / 数据源（如 Refinitiv、Polygon、Backed feed）取中位数。
盘后 / 周末：使用 last-print + 期权隐含价 + 链上 xStocks 报价拼接。指数实际上 24/7 不停，但盘后流动性差，价差可能 ±2% 跳动。

结算频率
三家交易所代币股永续大都沿用 8 小时结算：
  • 币安 / Bitget：00:00 / 08:00 / 16:00 (UTC) = 北京 08:00 / 16:00 / 00:00
  • 欧意：00:00 / 08:00 / 16:00 (UTC)，部分品种为 4 小时（标 funding interval）
  • 美股开盘前后（北京时间 21:30 / 04:00）经常出现费率剧烈波动，是套利入场窗口
实时倒计时见 "💰 资金费率" 标签
🧮
资金费率公式
F = clamp(Premium + clamp(InterestRate − Premium, ±0.05%), ±cap)
其中 Premium = (Mark − Index) / Index（按 TWAP 平均）。
cap：币安 / Bitget 默认 ±0.75%/8h，欧意默认 ±1.5%/8h；股票类品种可能调高至 ±3%/8h（财报前后）。
超 cap 的费率会被截断，溢价持续无法对齐
🚫
公司行动 (Corporate Action)
分红：合约通常 不分红，但会在除息日前调低指数（变相扣除）。
拆股 / 并股：按比例调整合约面值与持仓数量，相当于换股。
停牌 / 收购退市：交易所会强制下架并按最后指数 / 收购对价结算 — 持仓被强平的风险极高，建议事件窗口前清仓。
PATH A 与 PATH B 处理方式不同：xStocks 现货走链上 rebase
🏛️
xStocks 实物代币（Path B）
发行方：Backed Finance AG（瑞士持牌）。 托管：Clearstream / InCore Bank，1:1 持有真股票。
链：Solana 主网（部分桥接到 Base / BSC）。
申赎：仅限 KYC 机构，零售只能在交易所二级市场交易； 当 xStock 链上价格偏离美股 > 0.5% 时，授权参与者会做无风险套利拉回。
链上合约可在 Solscan 验证储备证明
📐
杠杆现货 = 借币 + 现货
做空逻辑：从交易所借入 xStock → 卖出换 USDT → 等价格下跌后买回归还。
成本：借币小时利率（年化 5-50%，看库存紧张度）+ 现货手续费 × 2。
注意 借币利率 ≠ 永续资金费率，两者独立浮动 — 这正是跨工具套利的来源。
Bitget 杠杆现货支持 NVDAX / TSLAX 等
什么情况下会有风险？ 按严重度排序 · 持仓前必读
核心原则：代币股是 "包了一层加密皮"的美股， 底层是 NYSE 时段定价，外层是 24/7 加密交易 —— 错配（Mismatch）就是所有风险的根源。 永远不要在不知道下一次交易时段开始时间的情况下重仓。
套利策略 · 资金费率驱动 仅供研究参考 · 自负盈亏
套利能否成立的前提：资金费率与结算时间数据 必须实时更新。 陈旧的费率 = 错误的入场点 = 套利失败。本页 "💰 资金费率" 标签每 60 秒自动刷新； 在大行情 / 财报前后建议手动 ↻ 刷新到秒级精度，再下单。
